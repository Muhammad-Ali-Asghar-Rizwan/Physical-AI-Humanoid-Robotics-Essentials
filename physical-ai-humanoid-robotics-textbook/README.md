# Physical AI & Humanoid Robotics Textbook

This repository contains the source code for the Physical AI & Humanoid Robotics textbook, built with Docusaurus for optimal web presentation and GitHub Pages deployment.

## About

This textbook explores the cutting-edge intersection of artificial intelligence and humanoid robotics, covering topics from biomechanics to cognitive systems and ethical considerations in AI development.

## Features

- Interactive documentation using Docusaurus v3
- GitHub Pages ready deployment
- Modular chapter structure
- Bonus features for enhanced learning experience

## Structure

- `docs/chapters/` - Textbook content chapters
- `specs/` - Specification documents for features
- `RAG-backend/` - Retrieval-Augmented Generation backend implementation
- `src/pages/` - Additional pages and bonus features

## Bonus Features

This textbook includes experimental features:
- Authentication system (bonus)
- Personalization engine (bonus)
- Urdu language translation (bonus)

## Deployment

This site is automatically deployed to GitHub Pages on pushes to the main branch.
